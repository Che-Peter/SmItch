
# Arduino Firmware (Mega Pro Mini)

This version is the original SmItch firmware.  
It runs on the Arduino Mega Pro Mini and provides:

- Offline scheduling using RTC  
- Current sensing with ACS712 (±5A)  
- Relay control via ULN2803 drivers  
- Safe start‑up and surge protection  

Relays are 12V to ensure stable switching and avoid fast voltage dropouts.
