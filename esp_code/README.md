# ESP8266 Firmware (Version 2)

This version extends SmItch with internet connectivity.  
It integrates with the Arduino base design and adds:

- Wi‑Fi control from anywhere in the world  
- Synchronization with RTC for schedules  
- Improved memory handling and responsiveness  

This firmware works alongside the Mega Pro Mini hardware, enabling hybrid offline + online control.

---

**Note:** This repository contains simplified firmware and documentation for demonstration purposes.  
**It is not the final production firmware. The complete project remains confidential and non‑disclosable.**  
**What is shared here represents a basic working version to illustrate the concept and core features of SmItch.**
